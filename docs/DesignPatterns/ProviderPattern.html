Provider Pattern

參考連結： React Context, Provider and useContext

此篇原文：Provider Pattern，詳細的介紹可以讀這篇文章，會有一個步驟一步驟的過程，並且搭配圖片做說明

前言​

當 Component 太多太深層，需要傳遞參數時，可能會從 Root 結點一路傳遞下去，造成 code 不好閱讀，Provider Pattern 可以透過 Provider 與 useContext hook ，讓資料存在外層的 Provider ，要使用的 component 可以使用 hook 來取得需要的資料

範例​

在沒有使用 Provider Pattern 時，props data 要傳遞到底層的 component ，只能一層一層傳，並且還有中間層是只為了傳遞 props 就需要引用並且 assign

Before Provider Pattern
function App() {
  const data = { ... }

  return (
    <div>
      <SideBar data={data} />
      <Content data={data} />
    </div>
  )
}

const SideBar = ({ data }) => <List data={data} />
const List = ({ data }) => <ListItem data={data} />
const ListItem = ({ data }) => <span>{data.listItem}</span>

const Content = ({ data }) => (
  <div>
    <Header data={data} />
    <Block data={data} />
  </div>
)
const Header = ({ data }) => <div>{data.title}</div>
const Block = ({ data }) => <Text data={data} />
const Text = ({ data }) => <h1>{data.text}</h1>

使用方式很簡單

  1. 使用 createContext 建立一個 Provider，並且包在最外層 Component 外面
  2. 使用 useContext 取得 Provider 的儲存的資料
Create Provider
const DataContext = React.createContext()

function App() {
  const data = { ... }

  return (
    <div>
      <DataContext.Provider value={data}>
        <SideBar />
        <Content />
      </DataContext.Provider>
    </div>
  )
}
Use Provider Data
const SideBar = () => <List />;
const List = () => <ListItem />;
const Content = () => (
  <div>
    <Header />
    <Block />
  </div>
);

function ListItem() {
  const {data} = React.useContext(DataContext);
  return <span>{data.listItem}</span>;
}

function Text() {
  const {data} = React.useContext(DataContext);
  return <h1>{data.text}</h1>;
}

function Header() {
  const {data} = React.useContext(DataContext);
  return <div>{data.title}</div>;
}

而在我在實務上，會將 Provider 與 Hook 包成一個檔案，方便使用

note

詳細可以參考上面連結的文章，會對 Provider 的使用更有想法與畫面

Provider file
import React, { createContext, useCallback, useContext, useMemo, useState } from 'react';

const DataContext = createContext(undefined);

export function DataProvider({ value, children }) {
  const [data, setData] = useState(value);

  const initData = useCallback(() => {
    setData({...});
  }, []);

  const contextData = useMemo(() => {
    return {
      data,
      initData,
    };
  }, [data, initData]);

  return (
    <DataContext.Provider value={contextData}>
      {children}
    </DataContext.Provider>
  );
}

export function useData() {
  const contextData = useContext(DataContext);

  if (contextData === undefined) {
    throw new Error('useData must be used within a DataProvider');
  }

  return contextData;
}

就可以透過取出 Provider 與 hook 來使用

import {DataProvider, useData} from '~/DataProvider'

function App() {
  const data = { ... }

  return (
    <div>
      <DataProvider value={data}>
        <SideBar />
        <Content />
      </DataProvider>
    </div>
  )
}

const SideBar = () => <List />
const List = () => <ListItem />
const Content = () => <div><Header /><Block /></div>

function ListItem() {
  const { data } = useData();
  return <span>{data.listItem}</span>;
}

function Text() {
  const { data } = useData();
  return <h1>{data.text}</h1>;
}

function Header() {
  const { data } = useData();
  return <div>{data.title}</div>;
}
